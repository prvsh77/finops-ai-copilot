# Product Features

FinOps AI Copilot provides a complete set of features to automate, secure, and project corporate financial operations.

## 1. Enterprise AI Copilot Console
* **SSE Token Streaming**: Real-time response generation displaying logical reasoning steps chunk-by-chunk.
* **Specialist Orchestrator**: Supervisor agent parses inputs, queries the RAG document index, fetches context from tool registries, and delegates specialized tasks.
* **Conversation History**: Full multitenant session memory retention, listing previous conversations and loading transcripts instantly.
* **Cost Telemetry**: Transparent tracking of tokens processed, request counts, latencies, and gateway fees.

## 2. AI Threat Scanner (Fraud Center)
* **Duplicate Traps**: Automated scanning for duplicate payments within a 7-day period and duplicate invoice numbers.
* **Anomaly Detection**: Weekend postings, high-value transfer triggers (> INR 50L), and blocked vendor verification locks.
* **Impact Cards**: Instant risk rating dashboards (Critical/Medium/Low) highlighting exact reasons and actionable mitigations.

## 3. Financial Analytics & Forecasting
* **Projections Matrix**: 6-month projected revenue, expense, and cash flow runways.
* **AI Commentary**: Direct, logical breakdown of working capital variance based on APAC and regional collections.
* **Variance Analytics**: EBITDA margin charts, budget burn thresholds, and category spend concentration charts.

## 4. Reports & Board Summary
* **Board Brief Executive Summary**: Automated high-level text brief summarizing EBITDA gains, unapproved AP balances, and critical warnings.
* **Report Library**: Auto-saved PDF-ready documents detailing department spend, variance calculations, and audit logs.
* **Export Engine**: Support for CSV, JSON, and Microsoft Excel data files download.

## 5. Global Search Engine
* **Unified Keyword Matches**: Single header search box instantly querying transactions, invoices, payments, vendors, customers, reports, and fraud alerts.
* **Dynamic Direct Linking**: Click search matches to hop straight to the relevant detail tab.
